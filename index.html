🌱 About GrowMap v0.3

GrowMap helps you discover which plants can thrive at any location on Earth. Click anywhere on the map to get real climate data and growability scores.

What's new in v0.3

  • 🔌 Perenual API — 10,000+ species with real images
  • 🌺 Trefle API — 5,000+ species as fallback
  • 🖼️ Plant images in cards and modals
  • 🧠 Smart dedup — combines all sources, keeps best data
  • ⚙️ API config panel — add your own keys in-app

Data Sources

  • 📍 Location search & reverse geocode via Nominatim
  • 🌡️ Climate data via Open-Meteo
  • 🧪 Soil pH via ISRIC SoilGrids
  • 🌿 Plants via Perenual & Trefle

Score Meaning

🟢 80-100 Excellent 🟡 60-79 Good 🟠 40-59 Possible 🔴 20-39 Difficult ⚫ 0-19 Not Recommended
